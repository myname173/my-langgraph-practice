# UE5 宣传网站 (UE5 Promo Site)

## 项目概述

这是一个纯静态的虚幻引擎 5 宣传网站，使用原生 HTML5、CSS3 和 JavaScript 构建，无需任何构建工具或依赖。

## 项目状态

✅ **已完成** - 所有核心功能已实现并测试通过

## 文件结构

```
ue5-promo-site/
├── index.html          # 主页面 (130 行)
├── css/
│   └── styles.css      # 样式文件 (338 行)
└── js/
    └── main.js         # 交互脚本 (252 行)
```

## 功能特性

### 1. 页面结构
- **导航栏**：响应式导航，包含 Logo 和 4 个导航链接
- **英雄区域**：展示主标题、副标题和行动号召按钮
- **功能特性区**：4 个功能卡片展示核心功能（实时渲染、高性能、蓝图系统、跨平台）
- **画廊展示**：3 个作品展示卡片，带悬停效果
- **下载区域**：下载按钮和版本信息
- **页脚**：包含关于、资源链接和联系方式

### 2. 交互功能 (main.js)
- ✅ **移动端菜单切换**：汉堡菜单按钮控制导航栏展开/收起
- ✅ **滚动导航栏效果**：页面滚动时导航栏添加阴影效果
- ✅ **平滑滚动**：点击导航链接平滑滚动到对应区域
- ✅ **IntersectionObserver 动画**：元素进入视口时触发动画效果
- ✅ **键盘导航支持**：支持 Tab 键导航和 Enter 键激活
- ✅ **懒加载**：图片资源按需加载
- ✅ **波纹效果**：按钮点击时的视觉反馈

### 3. 样式特性 (styles.css)
- ✅ **CSS 变量**：定义主题颜色和间距变量
- ✅ **基础重置**：统一的浏览器样式重置
- ✅ **按钮样式**：主按钮和次按钮两种风格
- ✅ **响应式布局**：
  - 移动端导航栏（汉堡菜单）
  - 功能卡片网格（单列/多列自适应）
  - 画廊图片自适应尺寸
  - 字体大小响应式调整
- ✅ **动画效果**：悬停效果、过渡动画、淡入效果

### 4. 响应式断点
- **768px**：平板设备适配
- **480px**：手机设备适配

## 技术栈

- **HTML5**：语义化标签结构
- **CSS3**：Flexbox 布局、CSS 变量、媒体查询、动画
- **JavaScript (ES6+)**：原生 JS，无第三方依赖
- **图片资源**：使用 placeholder 占位图（符合演示站点预期）

## 使用说明

### 本地运行
1. 直接在浏览器中打开 `index.html` 文件
2. 或使用任意静态文件服务器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (需安装 http-server)
   npx http-server
   ```

### 部署
由于是纯静态项目，可部署到任意静态托管服务：
- GitHub Pages
- Netlify
- Vercel
- 传统 Web 服务器 (Nginx/Apache)
